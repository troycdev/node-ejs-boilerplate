# Node EJS Boilerplate

### Description

Basic boilerplate project for getting started with Node.js projects using EJS.

### Usage

To get started, clone this repository and run `npm install` to install all required dependencies. Spin up a development server with `npm run dev`.

### Built With

- [EJS](https://ejs.co/) - Embedded JavaScript templating
- [Express](https://expressjs.com/) - Web framework
- [Mongoose](https://mongoosejs.com/) - Database modelling
- [dotenv](https://www.npmjs.com/package/dotenv) - Environment variable loader
- [Chalk](https://www.npmjs.com/package/chalk) - Terminal styling
